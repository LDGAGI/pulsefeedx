## 🧩 一、MVP期的核心目标

> 🎯 **目标：验证用户是否愿意“依赖”你的信号提醒。**

这一阶段不是做产品，而是造一个“能唤醒人的体验原型”。

衡量标准只有两个：

1. 用户第一次体验后说出“这真有用”。
2. 3天内用户还想回来继续用。

---

## ⚙️ 二、功能范围（极简可验证）

| 模块            | 功能描述                           | 用户感知           | 技术实现                         |
| ------------- | ------------------------------ | -------------- | ---------------------------- |
| **1️⃣ 监控输入**  | 用户输入一个关键词或推特账号                 | “我可以告诉系统我关心什么” | 简单网页表单 / JSON 配置             |
| **2️⃣ 实时监听**  | 调用 Twitter API Filtered Stream | “它真的在实时监控”     | Python + Flask + Twitter API |
| **3️⃣ 匹配过滤**  | 命中关键词→触发提醒                     | “只在关键时刻打扰我”    | 正则 / 简单 NLP 匹配               |
| **4️⃣ 通知推送**  | Telegram Bot 通知（秒级）            | “世界在动，我立刻知道”   | python-telegram-bot          |
| **5️⃣ 日志可视化** | 简单网页展示命中记录                     | “我能看到它真的在工作”   | Flask + Tailwind / CSV 表格    |

💡 保持单一闭环：**信号 → 推送 → 反应**
其他功能（注册、登录、AI摘要、历史记录）全部延后。

---

## 🧠 三、MVP运行逻辑

```
用户输入关键词或账号
     ↓
系统监听推文流
     ↓
命中规则触发通知
     ↓
用户收到推送
     ↓
收集反馈与反应（手动或日志）
```

💡 **关键体验点：** 推送延迟必须 < 5 秒。
速度感 = 信任感。

## 🧱 四、技术实现建议（快速落地栈）

| 环节          | 工具建议                          | 理由              |
| ----------- | ----------------------------- | --------------- |
| Twitter API | **Filtered Stream**           | 支持实时推文流监听       |
| 后端框架        | **Flask**                     | 快速搭建 webhook    |
| 推送模块        | **Telegram Bot API**          | 延迟低、免费、易测试      |
| 数据存储        | **JSON / SQLite**             | 不用上数据库          |
| 前端          | **Next.js / Tailwind**        | 简易 landing + 表单 |
| 部署          | **Render / Vercel / Railway** | 无需服务器维护         |

## 📊 六、用户验证方法

1️⃣ 找 5–10 位早期用户（AI 或加密领域活跃者）
2️⃣ 每人设置一个关键词（如 "launch" / "AI Agent"）
3️⃣ 在 48 小时内记录：

* 收到几次提醒
* 哪次提醒让他觉得有价值
* 是否主动添加新关键词
  4️⃣ 收集反馈关键词：

  * 出现 “快”、“准”、“救了我”、“神了”、“想多设几个” → ✅ PMF信号

## 💰 七、MVP成功标准（可量化）

| 指标          | 目标          | 意义     |
| ----------- | ----------- | ------ |
| 平均触发延迟      | <5 秒        | 核心体验达标 |
| 日活用户（10人样本） | >50%        | 有使用粘性  |
| 每用户平均关键词数   | ≥2          | 用户主动扩展 |
| 主观反馈        | 至少3人表达“依赖感” | 达到尖叫点  |
